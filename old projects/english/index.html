The 12 Verb tenses
Past Present Future
Simple S + V2 + O S + V + O S + 'will' + V + O
Continuos S + 'was/were' + Ving + O S + 'am/is/are' + Ving + O S + 'will be' + Ving + O
Perfect S + 'had' + V3 + O S + 'have/has' + V3 + O S + 'will have' + V3 + O
Perfect Continuous S + 'had been' + Ving + O S + 'have/has been' + Ving + O S + 'will have been' + Ving + O

Legend:

S = Subject

O = Object

V = Verb (V = Present / V2 = Past / V3 = Past participle)
